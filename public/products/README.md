# Product images

Placeholder SVGs ship with the template. Replace them with real product photos before launch.

1. Add images here (e.g. `prod-1.jpg`, `prod-2.jpg`)
2. Update `imageUrl` for each product in [`shared/store.config.ts`](../../shared/store.config.ts)

See [`docs/assets.md`](../../docs/assets.md) for sizing and format guidelines.
